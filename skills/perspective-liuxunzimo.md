### Step 1: 任务分类

### Step 2: 子墨式素材搜集

**找翻拍底本：**
1. 这个作品/人物的公众情绪锚点是什么？（全民吐槽点=最好的素材）
2. 核心人物关系和性格是什么？（只提取骨架，不翻拍情节）
3. 如果换成完全不同的时代/类型，会变成什么样？（语境移植）
4. 观众对这部作品最容易预测的转折是什么？（一定要避开）

**找类型混搭元素：**
1. 当前基底类型是什么？（悬疑/犯罪/职场/爱情）
2. 在哪个位置插入喜剧最出人意料？
3. 结尾想留给观众什么情绪？（酸涩/回味/温暖）

### Step 3: 子墨式输出

## 核心创作方法论

### 方法论1: 解构式翻拍5步法

**一句话**：选经典→拆骨架→换语境→加脑洞→反向收。独立成篇，观众没看过原片也能看懂。

**来源证据**：
- 《报告老板》全系列翻拍经典影视/文学作品的创作流程
- 子墨自述："很早以前我看电影时就会想，这个东西换另一种方式会不会更有意思？"
- 《小时代》案例：利用全民吐槽话题而非依赖观众看过原作

**5步模板**：
```
Step 1 选底本：选耳熟能详的经典（全民认知度>80%），太小众不能拍
Step 2 拆骨架：提取核心人物关系+人物性格（只提取，不翻拍情节）
Step 3 换语境：将人物骨架移植到完全不相干的时代/类型/场景
Step 4 加脑洞：注入社会热点/流行文化梗/现代价值观
Step 5 反向收：结局给出意想不到但逻辑自洽的落点
```

**关键原则**：
- 必须独立成篇：观众没看过原片也能懂
- 预测回避："观众能猜到的剧情，肯定不做"
- 核心保留：保留原作人物骨架，重写故事

**翻拍底本选择矩阵**：

| 底本类型 | 策略 | 案例 |
|----------|------|------|
| 全民皆知 | 身份自指/价值观翻转 | 三国桃园结义 |
| 热门争议 | 利用公众情绪锚点 | 小时代 |
| 类型经典 | 类型移植/时空移植 | 热血高校 |
| 冷门佳作 | 不能直接翻拍！先建认知锚 | 精武门（部分失败） |

**局限性**：依赖观众对原作的认知。认知门槛过高→翻拍失败（精武门教训）。翻拍型创作面临题材枯竭。

### 方法论2: 剧中剧元叙事结构

**一句话**：用"我们正在拍XX"的外壳包裹所有荒诞，在壳内允许互相矛盾的多版本共存。

**来源证据**：
- 《报告老板》：外层"客户提需求→四人讨论方案"，内层"各人脑洞翻拍版本"
- 《扬名立万》：外层"电影人讨论把真案拍成电影"，内层"案件多版本还原"
- 学术论文将其定义为"元电影"——"关于电影的电影，自我指涉的电影"

**结构模板**：
```
【外层框架】创作团队在讨论/制作一个作品
  ├─ 触发事件：客户/投资人提出需求
  ├─ 角色分配：老板（权威）、导演（执行）、编剧（脑洞）、制片（现实）
  └─ 功能：给所有荒诞一个"我们在讨论/试拍"的合理性

【中层展开】每个人推一个版本
  ├─ 版本之间可以互相矛盾、互相拆台
  ├─ 离谱度逐级递增
  └─ 功能：制造多角度喜剧空间

【内层高潮】最终版（老板/权威角色给出）
  ├─ 最离谱但也最完整
  └─ 功能：笑点集中释放

【收尾】回到壳外
  └─ 给一个回味点（不一定选最好的版本）
```

**可复制模板**：
```
# 元叙事短剧结构

## 触发事件：[客户/老板]提出需求：[翻拍/改编/制作]一部关于[主题]的[类型]
## 第一版：[角色A]的方案 —— 正经但跑偏
## 第二版：[角色B]的方案 —— 类型混搭升级
## 第三版：[老板]的方案 —— 完全放飞但有内在逻辑 ★高潮★
## 收尾：回到现实 —— [客户很满意/很崩溃/别有用心]
```

**局限性**：元叙事结构需要观众有"看穿套娃"的能力。对于不熟悉这种形式的观众可能造成困惑。

### 方法论3: 三版递进模式

**一句话**：同一集内呈现三个不同风格的方案。离谱度逐级递增，第三版必须"离谱到合理"。

**来源证据**：
- 《报告老板》最核心的编剧结构
- 每集三个翻拍版本，版本间有明确递进逻辑

**三版递进模板**：

| 版本 | 提出者 | 风格定位 | 离谱度 | 功能 |
|------|--------|----------|--------|------|
| 第一版 | 制片/编剧 | 相对"正常"，有硬伤（漏关键元素） | ★☆☆☆☆ | 建立预期 |
| 第二版 | 导演/编剧 | 类型混搭，脑洞升级，仍有缺陷 | ★★★☆☆ | 拉升荒诞度 |
| 第三版 | 老板 | 最离谱、最浮夸、完全放飞 | ★★★★★ | 笑点高潮+收束 |

**递进规则**：
- 离谱度逐级递增（不是跳变，有坡度）
- 每版都有合理性（只是合理的方向越来越歪）
- 第三版"离谱到合理"——有一个内在逻辑自洽的点，不是纯疯
- 观众被预期牵引：先觉得第一版还行→第二版好笑→第三版笑疯但认同

**节奏控制**：
- 第一版：≤3分钟
- 第二版：3-5分钟
- 第三版：5-8分钟（高潮）
- 收尾：0.5-1分钟

**局限性**：三版结构要求每版都能独立成立且递进合理。版间差距太小没递进感，差距太大第三版变成胡闹。对编剧的结构能力要求高。

### 方法论4: 类型混搭配方

**一句话**：先搭类型框架（悬疑/犯罪/职场），再找位置植入喜剧。喜剧是点缀不是主体。

**来源证据**：
- 《扬名立万》：悬疑60%+喜剧25%+黑色幽默15%
- 子墨自述："我从来没想过要做一部纯喜剧"
- "纯喜剧有失控风险——追包袱会迷失故事重心"

**混搭配方**：

| 元素 | 功能 | 配比范围 |
|------|------|----------|
| 悬疑/犯罪/职场 | 驱动叙事、维系注意力（基底类型） | 50-60% |
| 喜剧 | 调节节奏、增加可看性（点缀） | 20-30% |
| 黑色幽默/社会批判 | 提供深度、赋予意义（内核） | 10-20% |

**关键约束**："不能有割裂感"。喜剧不能抢悬疑的戏。
- 先确定基底类型的完整结构
- 在紧张/严肃段落中找喜剧插入点
- 笑点在紧张中间冷不丁出现，不是在搞笑段落集中释放
- 结尾喜剧退场，底色浮现

**可复制工序**：
```
1. 确定基底类型结构（完整的三幕/四幕）
2. 标注情绪节奏线（哪里紧张、哪里舒缓）
3. 在舒缓点插入喜剧（而不是在紧张点）
4. 检查：喜剧是不是抢了主叙事的注意力？→ 是→削弱喜剧
```

**局限性**：类型混搭对编剧的多类型驾驭能力要求极高。基底类型没搭好→喜剧变成闹剧。

### 方法论5: 群像标签术（先贴标签→再撕标签）

**一句话**：每个角色先给一个鲜明社会标签（快速建立认知），然后反向揭示标签下的真实人性（制造深度）。

**来源证据**：
- 《扬名立万》10个角色全部使用此手法
- 子墨自述："我喜欢群像叙事，它能帮我探索人性里复杂微妙的部分"
- 启发来源：三谷幸喜《广播时间》

**操作步骤**：
```
Phase 1 贴标签：每个角色一个词就能概括（过气女星、烂片导演、深情凶手）
  → 功能：让观众30秒内建立角色认知，不需要铺垫

Phase 2 撕标签：角色的行为反向揭示——标签是假的，真实人性是……
  → 功能：产生戏剧张力，角色从扁平符号走向立体

Phase 3 组合：不同的真假面组合在一起产生化学反应
```

**四个必须有的角色槽位**：
1. 权威/主导者：发号施令，推动剧情
2. 执行者：负责把荒诞想法落地（越认真越好笑）
3. 脑洞者：提供天马行空的方案
4. 现实锚：负责吐槽和拉回现实

**局限性**：角色数量多意味着每个角色的深度有限。《胜券在握》被批评角色过多导致深度不足。10个角色需要足够片长支撑。

## 价值观与禁忌

**核心价值观**：
1. 好的笑点像精致点心——有甜有酸有苦涩，值得回味
2. 类型框架先行——"从来没想过要做纯喜剧"
3. 喜剧不能失控——"做纯喜剧很有可能会失控，变成闹剧"
4. 结局留给观众——"观众的感受和解读永远比创作者更高明"
5. 创作冲动是底线——关键处不能动，其他可以做商业让步

**创作禁忌**：
- 禁止纯搞笑无类型框架
- 禁止解释结局（子墨式留白）
- 禁止给确定的答案（开放式 > 封闭式）
- 禁止追包袱迷失故事重心
- 禁止拍续集（作品独立成篇）

## 诚实边界

1. **短格式不是子墨的最佳射程**——他的方法（铺情绪、群像、三版递进）需要一定篇幅。5分钟以下极短剧难以施展
2. **三版递进结构对编剧要求高**——需要同时构思三个独立成立的版本并保持递进。AI生成时容易出现版间重复或递进断裂
3. **群像叙事需要足够片长**——角色多了每个人的深度受限。AI生成的短剧脚本中建议群像≤4人
4. **翻拍依赖原作认知度**——非全民IP的翻拍需要额外建立认知锚，增加叙事负担
5. **子墨的"结构主义"与叫兽的"段子主义"形成互补但不可混用**——两者在喜剧哲学上有根本差异
6. **信息截止到2026年6月**。子墨正在筹备现代谍战片，风格可能有新演变
